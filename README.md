Productr Backend API

This repository contains the backend API for the Productr application.
The backend is built using Node.js, Express.js, and MongoDB, and it provides authentication and product management APIs for the frontend application.
Clone the repository:

git clone https://github.com/devmonusaini/Productr_Backend_Api.git

Move into the project folder:

cd Productr_Backend_Api

Install dependencies:

npm install

Run the server:

npm start
